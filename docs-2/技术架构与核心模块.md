## 核心模块详解

### 模块一：数据处理模块 (src/data/)

**出处**: src/data/loader.py, src/data/preprocessing.py, src/data/time_alignment.py

| 子模块 | 功能 |
|--------|------|
| DataLoader | 支持CSV/Excel/JSON/Parquet格式加载 |
| DataPreprocessor | 缺失值处理、异常值检测、队列筛选 |
| TimeAligner | 相对时间轴构建、时间窗口归并 |

### 模块二：特征工程模块 (src/features/)

**出处**: src/features/symptom_extractor.py, src/features/medicine_mapper.py, src/features/syndrome_encoder.py

| 子模块 | 功能 | 知识库规模 |
|--------|------|-----------|
| SymptomExtractor | 症状提取+严重程度量化 | 60+症状，3级编码 |
| MedicineMapper | 药物→功效类别映射 | 100+药物，10大功效 |
| SyndromeEncoder | 证型→多热编码 | 支持多证候 |

### 模块三：因果推断模块 (src/causal_inference/)

**出处**: src/causal_inference/propensity_score.py, src/causal_inference/ate_estimator.py

| 子模块 | 功能 | 支持方法 |
|--------|------|----------|
| PropensityScoreMatcher | 倾向性评分模型 | 逻辑回归、随机森林 |
| matching | 患者匹配 | 最近邻匹配 |
| ATEEstimator | ATE估计 | 均值差、回归调整、IPW、双鲁棒 |
| validator | 平衡性检验 | 标准化差异 |

### 模块四：因果发现模块 (src/causal_discovery/)

**出处**: src/causal_discovery/algorithms.py, src/causal_discovery/bootstrap.py

| 子模块 | 功能 | 实现 |
|--------|------|------|
| CausalDiscovery | PC/FCI算法 | causal-learn库 |
| StabilitySelector | Bootstrap稳定性选择 | 1000次重采样 |
| ConstraintManager | 约束管理 | 时序+领域知识 |

### 模块五：可视化模块 (src/visualization/)

**出处**: src/visualization/dag_plotter.py, src/visualization/report_generator.py

| 子模块 | 功能 | 输出 |
|--------|------|------|
| DAGPlotter | 绘制因果有向无环图 | PNG/PDF |
| ReportGenerator | 生成分析报告 | Markdown |
