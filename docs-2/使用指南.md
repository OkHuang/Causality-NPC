# 使用指南与简历建议

## 快速开始

### 环境准备

```bash
# 1. 创建虚拟环境
python -m venv venv
source venv/bin/activate  # Linux/Mac

# 2. 安装依赖
pip install -r requirements.txt

# 3. 配置环境变量
cp .env.example .env
```

## 使用指南

### 目标一：验证性因果推断

**适用场景**: 想要验证"某种药物是否对某个症状有效"

**一键运行**:
```bash
python experiments/example_causal_inference.py
```

**输出文件**:
- `outputs/reports/causal_inference_report.md` - 分析报告
- `outputs/figures/balance_check.png` - 平衡性图表

### 目标二：因果发现

**适用场景**: 探索变量间的因果关系、构建因果网络图

**一键运行**:
```bash
python experiments/example_causal_discovery.py
```

**输出文件**:
- `outputs/graphs/causal_dag.png` - 因果网络图
- `outputs/reports/causal_discovery_report.md` - 发现报告

## 常见问题

### Q: 如何添加新的症状？

编辑 `docs/knowledge_base/tcm_symptoms.md`

### Q: Bootstrap太慢怎么办？

在 `config/base.yaml` 中降低 `n_bootstrap` 次数

### Q: 如何解读因果图？

- **节点**: 变量（症状/药物/证型）
- **有向边**: 因果关系（A → B 表示A导致B）
- **边的粗细**: 置信度（Bootstrap频率）
