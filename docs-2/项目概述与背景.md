# Causality-NPC 项目概述与背景

## 项目基本信息

**项目名称**: Causality-NPC (鼻咽癌因果推断项目)

**项目全称**: 中西医结合诊疗的因果推断与动态网络发现

**项目类型**: 医疗数据科学研究项目 / 因果推断研究

**项目角色**: 独立负责人（全权负责项目设计、开发与实施）

**项目周期**: 2026年1月 - 2026年2月（持续进行中）

**项目规模**: 约5000行Python代码，20+核心模块

---

## 项目背景与研究动机

### 医疗领域痛点

鼻咽癌（NPC）的中西医结合诊疗积累了海量数据，但"疗效评价"长期停留在**相关性分析**层面。临床实践中迫切需要回答因果性问题：

- **吃了这味药，患者症状是否真的有改善？**
- **哪些因果路径支撑着中医"辨证论治"的理论？**
- **如何从观察性数据中剥离混杂因素，量化药物的真实疗效？**

---

## 研究目标

本项目设计了**两大核心研究目标**，分别对应"验证性因果推断"和"探索性因果发现"：

### 目标一：验证性因果推断（模拟RCT）

**核心任务**：量化特定药物/疗法对症状的因果效应

**研究问题示例**：
- "补气活血药"是否显著改善了患者的"乏力"症状？
- 如果改善了，效应大小是多少（ATE）？

**技术路线**：
- 倾向性评分匹配（Propensity Score Matching, PSM）
- 模拟随机对照试验（RCT）的设计思路
- 平均处理效应（ATE）估计
- 平衡性检验与敏感性分析

### 目标二：探索性因果发现（因果网络构建）

**核心任务**：构建"症状-药物-证型"的动态因果网络

**研究问题示例**：
- "气虚证 → 补气药 → 乏力改善"这条因果路径是否存在？
- 中医诊疗中的隐变量和因果关系如何相互作用？

**技术路线**：
- PC/FCI算法（基于约束的因果发现）
- Bootstrap稳定性选择（1000次重采样）
- 时序约束与领域知识约束
- 因果路径提取与可视化

## 数据概况

### 数据集特征

- **数据来源**: 鼻咽癌患者中西医结合诊疗真实世界数据
- **记录规模**: 963条就诊记录
- **患者数量**: 约60-100名患者（存在多次随访）
- **时间跨度**: 2011-2024年（主要集中在2023-2024）
- **数据结构**: 纵向面板数据（Longitudinal Panel Data）

### 核心数据字段

| 字段名 | 类型 | 说明 | 示例 |
|--------|------|------|------|
| `patient_id` | integer | 患者唯一标识（用于纵向追踪） | 62760820 |
| `time` | date | 就诊日期 | 2024/3/11 |
| `chief_complaint` | text | 主诉文本（50-300字符） | 鼻咽癌放化疗后10年余，头痛明显... |
| `chinese_diagnosis` | text | 中医证候（可能多个） | 气虚毒热证 瘀血阻络证 |
| `chinese_medicines` | text | 中药处方（5-30味） | 黄芪 白术 丹参... |
| `age` | integer | 年龄 | 33-77 |
| `gender` | string | 性别 | 男/女 |

## 核心技术栈

### 数据处理与特征工程
- **基础库**: NumPy, Pandas, SciPy
- **机器学习**: Scikit-learn
- **文本处理**: 规则匹配 + LLM（OpenAI/Anthropic API）

### 因果推断核心算法
- **因果发现**: causal-learn（PC/FCI算法）
- **因果效应估计**: DoWhy, CausalML
- **统计检验**: statsmodels, pingouin

### 可视化与报告
- **图表绘制**: Matplotlib, Seaborn, Plotly
- **网络可视化**: NetworkX, Graphviz
- **报告生成**: Markdown自动生成

---

## 技术难点与解决方案

| 技术难点 | 解决方案 |
|---------|---------|
| 样本量小导致因果图不稳定 | Bootstrap稳定性选择（1000次重采样，保留频率>85%的边） |
| 缺乏因果真值验证 | 融入中医先验知识作为硬约束 + 专家盲测验证 |
| 非结构化文本特征提取 | 规则匹配 + LLM实体抽取（60+症状词典，100+药物映射） |
| 不固定时间间隔 | 时间轴对齐 + 相对时间窗口归并 |
| 混杂因素控制 | 倾向性评分匹配 + 平衡性检验（标准化差异<0.1） |
