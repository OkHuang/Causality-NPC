# 鼻咽癌中西医结合诊疗数据集说明

## 数据集概述

本数据集为鼻咽癌（NPC）患者中西医结合诊疗的**纵向随访数据**，记录了患者在不同时间点的就诊信息，包括西医诊断、中医辨证、治疗方案（中药、针灸）、症状、检验指标等。

**数据类型**：纵向面板数据（Longitudinal Panel Data）
**记录数**：963条
**文件格式**：CSV
**编码**：UTF-8

---

## 数据结构特点

### 1. 纵向随访结构
部分患者（`patient_id`）会有多次就诊记录（多条行），每条记录代表一次就诊事件。另一部分患者仅有1次就诊记录

**示例**：
```
patient_id    time           chinese_medicines
12345         2023/08/24     ["黄芪", "白术", ...]
12345         2023/10/18     ["黄芪", "白术", "丹参", ...]
12345         2024/01/08     ["党参", "茯苓", ...]
```

### 2. 多模态数据
- **结构化字段**：年龄、性别等（可直接使用）
- **半结构化字段**：检验指标（需解析）
- **非结构化文本**：主诉、中药处方（需NLP处理）

### 3. 时序因果关系
数据支持因果推断的关键特征：
- ✅ 有明确时间戳（`time`字段）
- ✅ 可追踪同一患者状态变化（`patient_id`）
- ✅ 记录了"症状→诊断→治疗"的完整链条

## 字段说明

### 标识字段

| 字段名 | 类型 | 说明 | 示例 |
|--------|------|------|------|
| `id` | integer | 记录唯一标识 | 91, 88, 68 |
| `checkup_id` | integer | 就诊次序标识 | 6276082005 |
| `patient_id` | integer | 患者唯一标识（用于纵向追踪） | 62760820 |

**注意**：同一`patient_id`对应多条记录 = 该患者的多次就诊

---

### 基本信息

| 字段名 | 类型 | 说明 | 取值范围/示例 |
|--------|------|------|--------------|
| `gender` | string | 性别 | "女", "男" |
| `age` | integer | 年龄（岁） | 33-77 |
| `time` | date | 就诊日期 | "2024/3/11", "2023/8/28" |

---

### 临床信息字段

#### 1. `chief_complaint`（主诉）
**类型**：文本（自然语言描述）
**内容**：患者的主要症状、病史、治疗史、当前症状等

**典型结构**：
- 肿瘤诊断时间、分期、治疗史（手术、放化疗）
- 当前主要症状（头痛、鼻塞、血瘀等）
- 生活质量相关（睡眠、食欲、体力等）
- 发病时间、频率、严重程度

**文本长度**：50-300字符

**示例格式**：
```
"鼻咽癌放化疗后10年余(2012年10月放疗结束,中肿)精神可,无恶风,怕冷,无头痛,头晕,咽干严重,无痰,无咳嗽,无鼻塞,喷嚏,流白清涕,无涕血,无耳鸣听力下降,无耳胀闷阻塞感,无耳流脓,无胃胀反酸,胃纳可,睡眠差,难入睡,易醒,多梦,夜尿1-2次,大便正常.舌硬活动不利.伸舌偏右,言语不利,吞咽欠顺畅."
```
#### 2. `chinese_diagnosis`（中医诊断）
**类型**：文本（多证候）
**内容**：中医证候诊断，可能包含多个证型

**示例格式**：
```
"气虚毒热证 瘀血阻络证"
"脾虚湿盛证 气滞血瘀证"
```

**特点**：
- 同一患者可能有多个证候（并列）
- 证候名称相对规范（基于中医理论）

#### 3. `western_diagnosis`（西医诊断）
**类型**：文本
**内容**：
- 并发症

**示例格式**：
```
"鼻咽恶性肿瘤 舌下神经麻痹 高脂血症 变应性鼻炎"
```

---

### 治疗信息字段

#### 4. `chinese_medicines`（中药处方）
**类型**：文本（药物列表，空格分隔）
**内容**：本次就诊开具的中药组成

**格式**：
```
"当归 黄芪 白术 茯苓 甘草..."
```

**特点**：
- 药物名称为中文
- 药物数量：5-30味不等
- 同一药名在不同处方中可能重复出现


#### 5. `western_medicines`（西药治疗）
**类型**：文本
**内容**：患者所服用的西药
**注意**：本数据集中该字段可能为空或较简略

---

### 中医诊断信息

#### 7. `tongue_condition`（舌诊）
**类型**：文本
**内容**：舌质颜色、舌苔厚度、舌下络脉等

**典型描述**：
```
"舌淡红，苔薄白"
"舌暗红，苔黄腻"
```

#### 8. `pulse_condition`（脉诊）
**类型**：文本
**内容**：脉象特征

**典型描述**：
```
"脉细数"
"脉沉涩"
```

#### 9. `physical_examination`（体格检查）
**类型**：文本
**内容**：西医体格检查发现（淋巴结、神经系统等）

**示例文本**：
```
舌肌萎缩，咽稍充血。双耳道正常，鼓膜完整，标志清。双下鼻甲肿大，色红，鼻咽稍充血，咽充血。会厌及声带正常。
```
---

### 检验指标

#### 10. `test`（实验室检查）
**类型**：半结构化文本
**内容**：血液学、生化、肿瘤标志物等检验结果

**格式**：
```
"2023-09-11,白细胞计数(WBC),5.6,×10^9/L
2023-09-11,血红蛋白(HB),120.0,g/L
2023-09-11,EB病毒DNA,<1000,copies/mL"
```

**特点**：
- 包含检验日期、指标名、数值、单位
- 需要正则表达式解析
- 指标数量：5-20个不等

## 数据示例

### 示例1：早期患者，多次随访

```csv
id,checkup_id,patient_id,gender,age,time,chinese_diagnosis,chinese_medicines,acupuncture,tongue_condition,pulse_condition,chief_complaint,test,is_last_checkup
91,6276082005,62760820,女,61,2024/3/11,"气虚毒热证 瘀血阻络","黄芪 白术 茯苓 甘草 丹参 川芎","百会 印堂 合谷","舌淡红苔薄白","脉细","鼻咽癌放疗后11年，近期头痛头晕，纳差，乏力",,no
88,6276082002,62760820,女,61,2023/8/28,"气虚毒热证","黄芪 白术 茯苓 甘草","百会 印堂","舌淡红苔薄白","脉细","鼻咽癌放疗后10年，头痛改善",,no
```

**说明**：
- 同一患者（patient_id=62760820）的两次就诊
- 时间跨度：2023/8/28 → 2024/3/11（约6个月）
- 证候变化：增加了"瘀血阻络"
- 处方调整：第2次加了丹参、川芎（活血药）
- 症状变化：头痛改善，但仍有头晕、乏力

## 数据质量说明

### 完整性
- ✅ 核心字段（`patient_id`, `time`, `chinese_diagnosis`, `chinese_medicines`）基本完整
- `western_medicines`部分空缺表示医生没有开西药
- ⚠️ `test`字段约50-70%非空

### 数据类型分布
| 类型 | 字段数 | 示例 |
|------|--------|------|
| 结构化（直接可用） | 5个 | id, age, gender, time, is_last_checkup |
| 分类文本（需编码） | 4个 | chinese_diagnosis, tongue_condition, pulse_condition, western_diagnosis |
| 自由文本（需NLP） | 3个 | chief_complaint, chinese_medicines, acupuncture |
| 半结构化（需解析） | 1个 | test |

### 时序特征
- **就诊间隔**：不固定，从1周至数月不等
- **随访次数**：每患者1-5次
