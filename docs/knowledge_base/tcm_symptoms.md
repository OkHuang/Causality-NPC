# 中医症状词典

## 使用说明

此文件定义了需要从主诉文本中提取的症状列表。

## 症状分类

### 耳鼻症状
- 耳鸣
- 听力下降
- 耳聋
- 鼻塞
- 鼻衄（鼻出血）
- 回吸性涕血
- 鼻痒
- 喷嚏

### 全身症状
- 乏力
- 消瘦
- 体重下降
- 发热
- 盗汗
- 畏寒
- 自汗

### 头颈部
- 头痛
- 头晕
- 颈部肿块
- 颈部疼痛

### 消化道
- 恶心
- 呕吐
- 纳差（食欲不振）
- 口干
- 口苦
- 腹胀
- 便秘
- 腹泻

### 精神睡眠
- 失眠
- 多梦
- 嗜睡
- 焦虑
- 抑郁
- 烦躁

### 呼吸道
- 咳嗽
- 咳痰
- 气促
- 胸闷

### 其他
- 面色苍白
- 面色萎黄
- 面色晦暗
- 舌质淡
- 舌质红
- 舌苔薄白
- 舌苔黄腻
- 脉细
- 脉沉
- 脉数

## 严重程度分级

- **0级（无/改善）**：无XX、XX改善、XX消失、XX缓解
- **1级（轻度）**：偶有XX、轻度XX、XX稍、XX微
- **2级（重度）**：XX明显、剧烈XX、严重XX、XX难以忍受、XX持续

## 注意事项

1. 此词典可根据实际数据情况进行补充
2. 症状提取支持正则表达式和LLM两种方式
3. 严重程度判定可通过修改配置文件中的关键词进行调整
