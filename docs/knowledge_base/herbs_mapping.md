# 中药功效映射表

## 使用说明

此文件定义了具体中药到功效类别的映射关系。

## 注意事项

1. 剂量为临床常用量，可根据实际情况调整
2. 某些药物需要特殊煎煮方法（先煎、后下等）
3. 一味中药可能具有多种功效，此处按主要功效归类
4. 如有新的药物需要添加，请遵循相同的格式
