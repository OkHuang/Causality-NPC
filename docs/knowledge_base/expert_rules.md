# 专家规则（因果约束）

## 使用说明

此文件定义了因果发现过程中需要遵守的领域知识约束。

## 时序约束

### 规则1：禁止未来影响过去
- **描述**：t+1时刻的变量不能指向t时刻的变量
- **实现**：通过变量名的时间标签自动检测
- **示例**：
  - 允许：`S_乏力_t -> M_补气药_t`（症状导致用药）
  - 禁止：`S_乏力_t1 -> M_补气药_t`（未来症状影响过去用药）

### 规则2：禁止药物瞬时起效
- **描述**：中药发挥作用需要时间，不能在同一时刻产生因果效应
- **实现**：禁止 `药物_t -> 症状_t` 的边
- **示例**：
  - 禁止：`M_补气药_t -> S_乏力_t`（同时间）
  - 允许：`M_补气药_t -> S_乏力_t1`（滞后效应）

## 领域知识约束

### 规则3：症状不能指向药物
- **描述**：症状可能导致医生开药，但从因果机制上，药物才是导致症状改善的原因
- **实现**：禁止 `S_* -> M_*` 的边
- **理由**：避免因果方向错误

### 规则4：人口学变量不随时间变化
- **描述**：年龄、性别等人口学特征是时不变的
- **实现**：这些变量不参与时序因果推断
- **变量**：`age`, `gender`

### 规则5：证型相对稳定
- **描述**：中医证型在短期内相对稳定，不会频繁变化
- **实现**：证型到证型的因果边需要谨慎验证
- **注意**：长期随访中证型可以演变

## 生物学不可能约束

### 规则6：治疗不能反向导致疾病
- **描述**：治疗手段不能是导致疾病的原因
- **实现**：禁止 `M_*_t -> D_*_t`（药物不能导致t时刻的证型）

### 规则7：西医治疗与中药的顺序
- **描述**：放疗/化疗通常在中药治疗前或同时进行
- **实现**：如果有时间信息，需遵循实际治疗顺序

## 因果强度约束

### 规则8：罕见组合谨慎处理
- **描述**：对于样本量<10的药物组合，不建立因果边
- **实现**：在特征降维阶段过滤

### 规则9：效应量阈值
- **描述**：只有效应量>0.1（标准化）的因果关系才保留
- **实现**：在后处理阶段过滤

## 注意事项

1. 这些规则是"软约束"，某些情况下可能有例外
2. 专家规则需要与中医专家反复讨论确认
3. 规则集可以随着对数据理解的加深而动态更新
4. 规则之间的冲突需要通过优先级解决
